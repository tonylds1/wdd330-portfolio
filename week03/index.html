Week03 - Notes

  • About Calling Methods

    object.method() is equivalent to object['method']()

  • About properties

    use the method hasOwnProperty for checking if a property exists.

    each property is like an element of an array/dictionary that have a key and a value.

    we can add and remove properties on the fly. It works even with those declared with const.

  • JSON vs literal notation

    for Json: double-quoted property names, restricter values allowed and methods not allowed

  • Built-in Objects

    Math is a built in object that brings many mathmaticals skills.

    Another Built-in objects: Date, RegExp

  • About DOM

    document is the root object.

    class, name and ids could be used to retrive html elements

  • Element Searching

    querySelector and querySelectorAll look for css selectors to retorn eleents

    ATENTION: when we are looking for an element and there is more than one option for the query, the querySelector will return just the last one.

  • About Camel Case Properties

    dashes are not leval characters in property names and should be turn into camel case.

  • Keyboards and Mouse events

    keydown, keypress, keyup and many others allow us to capture keyboards typing events. The same could be done with the mouse but the events names are different.

  • Removing Event Listeners

    it is useful to unbind events already binded.

  • Event Propagation

    An event is propagated throught its parent elements
